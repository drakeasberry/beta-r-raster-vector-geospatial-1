# NEON-R-spatial-raster
Lessons on using raster data in R, created through a collaboration between
Data Carpentry and NEON.

# Build lessons from Rmd

`make lesson-md`

# Serve jekyll locally

`make serve`

or

`bundle exec jekyll serve`
